
# TUIChat

TUIChat is a chat client for twitch made with terminal UI elements!



## Installation

No real installation needed.

```bash
  git clone https://github.com/Flyken27/TUIChat
  cd /tuichat

  edit the .env file with your information and desired channel

  use "https://twitchapps.com/tmi/" to generate yourself an oauth token

  run:
  go run .

  or 

  build & run:
  go build .

  then run exe

```
